## Introduction

_Color Calculations_ is a collection of helper functions for inspecting and manipulating CSS colors. You can drop this package into your own project to save you time and effort.

## Installation

Run this command to add this library to your `composer.json` file:

    composer require ganbarodigital/php-mv-color-calc

## Usage

See [our docs page](http://ganbarodigital.github.io/php-mv-color-calc) for a tutorial and reference documentation.
